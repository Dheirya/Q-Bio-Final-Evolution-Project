# QBio Final Evolution Project
A stochastic simulation of evolution for Columbia SHP Quantitative Bio Final Project. We used this code to determine the effects of a varying mutation rate on evolutionary success.

Our *full presentation* is on YouTube: **https://youtu.be/vLIabCkz0N4**
